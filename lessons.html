Lessons Learnts

The WO provides detailed metadata that compily to the DCAT vocabulary. As a result users have to fill a long form to list a dataset/app.

To enforce security and reduce barriers of data accessing we provide a single API for several types of datasets. The API unifies different dataset interfaces but no one is using it since is not compatibile with existing apps. We are developing a new TCP proxy-based technology that is able to transparently expose the same interfaces as underline databases while still protect them from unauthorised access.

It is still not straightforward for non-technicall users to fully exploit the Web Observatory.

The Web Observatory is not recognised by major journals yet.
